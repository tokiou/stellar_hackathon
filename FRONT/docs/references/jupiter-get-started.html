Why Jupiter

Jupiter is the liquidity infrastructure behind the majority of DeFi on Solana. Whether you’re building a trading terminal, a DeFi protocol, a payments app, or an AI agent, Jupiter gives you the APIs to ship fast and at scale. The same infrastructure that powers jup.ag is available to you as production-grade REST APIs. No RPC nodes, no blockchain state management, no transaction complexity. Jupiter abstracts all of it so you can focus on your product. Every API returns clean JSON, works with a single API key, and is designed to be consumed by both developers and AI agents.

Products

Swap

Token swaps with managed execution (/order) or raw instructions for custom transactions (/build).

Tokens

Search, metadata, verification status, organic score, and trading metrics for any Solana token.

Price

Heuristics-based USD pricing for up to 50 tokens per request.

Lend

Earn yield on deposits, borrow against collateral, and execute flashloans.

Trigger

Vault-based limit orders with single, OCO (TP/SL), and OTOCO order types.

Recurring

Automated dollar-cost averaging with time-based recurring orders.

Prediction

Binary prediction markets for real-world events.

Perps

Leveraged perpetuals trading on Solana (on-chain program).
